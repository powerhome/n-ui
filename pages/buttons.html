Use of Buttons

Buttons are arguably a design system\'s most important component. The use of this guideline is to help the Nitro team have an understanding to the different button uses through out Nitro. If used correctly, it will create a more comprehensive interface that the end use can understand from instinct.

Guidelines

Therse guidelines are the framework upon which we have built our system for how buttons are used in Nitro.
